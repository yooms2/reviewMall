# RM

- 인원 : 1명
- 기간 : 2023.03.20 ~ 2023.03.31
- DBMS : Oracle 11g Express Edition
- 언어 : Java SE 8 Platform, HTML5, CSS, JavaScript
- 디자인 패턴 : MVC
